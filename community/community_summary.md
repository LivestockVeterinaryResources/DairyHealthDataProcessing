# community summary

> **status: draft for community review.** the documents below are proposed by
> the founding contributors and are meant to be discussed, amended, and ratified
> by the community. items marked `<TODO>` still need confirmation.

this folder holds the documents that define our open, collaborative community —
who we are, how we make decisions, how to take part, and how we accept support
without compromising independence. the goal of the whole project is to improve
**cattle health and welfare** through better, data-driven decisions, built as
open, reproducible, community-owned tooling.

## the documents

| file | what it covers |
|---|---|
| [charter.md](charter.md) | our mission, scope, core values, and the guiding principle that **influence is earned through contribution, never purchased**. start here. |
| [governance.md](governance.md) | how decisions are made: merit-based maintainers, a steering committee with a **one-third cap on any single organization**, independent review of methodology, conflict-of-interest recusal, distributed copyright, and the permanent right to fork. |
| [contributing.md](../.github/contributing.md) | how to take part — code *and* non-code (expertise, data, docs, teaching). covers the contribution workflow, the developer certificate of origin (DCO) sign-off, and the `Supported by:` commit trailer. |
| [code_of_conduct.md](../.github/code_of_conduct.md) | the behavior we expect of everyone in community spaces (adapted from the contributor covenant 2.1). |
| [sponsorship.md](sponsorship.md) | how organizations can contribute resources on equal, transparent terms — **support buys capacity, not control.** |
| [supporters.md](supporters.md) | the organizations supporting the project, listed with equal weight, plus the independence statement. |
| [disclosures.md](disclosures.md) | the conflict-of-interest register where maintainers and decision-makers disclose relevant commercial relationships. |
| [citation.cff](../CITATION.cff) | machine-readable metadata so the project can be cited. |
| [license](../LICENSE) | the project's open-source license. |

## the short version

- **open by default** — open license, open development, open decisions; the right
  to fork is permanent.
- **vendor- and data-source-neutral** — the code serves the science and the
  animals, not any company's products.
- **everyone is welcome** — veterinarians, researchers, producers, analysts,
  educators, and students; expertise and resources alike.
- **industry support is welcome and safe** — accepted on equal terms, fully
  disclosed, and structurally prevented from biasing analysis or capturing the
  project.
